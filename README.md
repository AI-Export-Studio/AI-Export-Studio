# AI Export Studio 🚀

The elegant way to save ChatGPT & Gemini conversations to PDF, Word, and Markdown. No copy-paste. No mess.

## Features
- **Flawless Fidelity:** LaTeX, tables, and code blocks render exactly as they do in your browser.
- **Total Privacy:** Your chats never leave your machine. Everything is processed locally.
- **Selective Focus:** Cut out the noise. Select only the specific prompts and responses you need.
- **Deep Capture:** Automatically scrolls and captures massive conversations without missing a detail.
